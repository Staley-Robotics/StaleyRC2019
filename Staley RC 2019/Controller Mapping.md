# Drive Controller
- RT and LT
-Drive forward and back (Forward is the hatch side)
- B
-Gear Shift
- A
-Toggle rear lifter
- Y
-Toggle front lifter
- LB
-Vision Turning

# Alt Controller
- RT
-Move Hatch down
- LT
-Move Hatch up
- X
-Hatch pistons toggle
- A
-Intake
- B
-Shoot Balls
- Left Joystick
-Moves Shooter

## DPad/pivotTargets
- Start -Ground Position
- Down -Low Position
- Right -Mid Position
- Left -Cargo Position
- Up -High Position
